

### Parameters



|		Parameter 		   |	Default		|   Description    |
|--------------------------|:--------------:| ----------------:|
|stack 					   |		—		| Name of the stack|
|process name   		   | 	—			| Name of the target process |
|server name (optional)	   | 	—			| Name of the target server |
|count 					   |		—		| Desired relative count or absolute count (i.e. [+2],[-3] or 5) |
