### Parameters



    

        

            
Parameter

            
Description

        

    

    

        

            
_stack_

            
Name of your stack

        

        

            
_server name_ (optional)

            
Name of the server to access

        

        

            
_server ip_ (optional)

            
IP of the server to access

        

        

            
_server role_ (optional)

            
Role of the server to access (eg. web)
